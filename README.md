## 🚀 Project Overview
Attendify is a web-based **Student Management System** built using Django.  
It simplifies student record handling and attendance tracking through a clean UI and efficient backend processing.

---

## ✨ Key Features
- 👨‍🎓 Student Registration & Management  
- 📅 Attendance Marking System  
- 📊 View Attendance Records  
- ⚙️ Admin Panel for Data Management  
- 🔄 Full CRUD Operations

## ▶️ Getting Started

### 🔹 1. Clone Repository
bash
git clone https://github.com/Samrudhi1004/Attendify.git
cd Attendify
🔹 2. Install Dependencies
pip install -r requirements.txt
🔹 3. Run Server
python manage.py runserver

👉 Open in browser: http://127.0.0.1:8000/
